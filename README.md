## Abstract

**Primal** introduces a deterministic methodology utilizing prime number frequencies to generate high-dimensional embeddings. By leveraging the fundamental properties of prime numbers, this framework bridges the gap between **static hashing** (optimizing for orthogonality) and **dynamic manifold learning** (optimizing for reconstruction and invertibility). Unlike stochastic methods, Primal provides a theoretically grounded, deterministic approach to high-dimensional mapping.
